# HiWeekend_android
嗨周末是一款“私人定制”的周末出行信息推荐APP，在客户端首页提供各种信息推荐，内容涵盖运动、话剧、音乐会、读书会、展览、户外、酒吧、购物、电影等，部分活动在 APP 上预定会带来令人惊喜的超低折扣。
